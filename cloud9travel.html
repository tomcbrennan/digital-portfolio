Full-Stack Web Application

What is Cloud 9 Travel?

Cloud 9 Travel provides users with the ability to create an account, securely log in, and build their own virtual list of desired accomodation for future travels. Built with HTML, CSS & Javascript, this project was my first attempt at creating a single-page application with relevant and modern web development technologies. Cloud 9 Travel also boasts a built in backend database which makes use of Node JS and Express, with MongoDB for data storage. Further features of the website include hotel favouriting and password encryption with json web tokens.

Project Goals

Front-End Development

This project brought a great opportunity to familiarise myself with modern web development trends. In particular, the use of Single Page Applications (SPA), whereby a single HTML file is all that is needed for Javascript to manipulate the Document Object Model (DOM) and add or remove content within the page. SPA's are particularly useful for faster page-to-page content loading and a more dynamic application feel.

Back-End Development

Developing a back-end database was a crutial component of this project. With no previous experience in this form of development, I approached this task with a keen intent to learn as much as possible about the structure and function of a backend API. Using Express as the backend framework, this ensured the use a more adaptable format to object-orientated Javascript programming.

Full-Stack Functionality

A modern full-stack website application encompasses the power of technology in being able to handle user requests and dymanically communicate between the frontend interface and a backend server. In designing Cloud 9 Travel, it was essential that users were able to create their own customised hotels list as well as log in and out of their account, whilst also retaining their previous hotels and favourites.

What did I learn?

This project introduced me to the power of Javascript as a modern programming language as well as full-stack web development in general. Never before had I immersed myself so much in the area of web development, and I absolutely enjoyed every second of this project. Building an application such as this opens the doors to countless possibilities for future iteration and serves as a template for similar future projects. CRUD functionality is a key learning point for all web developers to take that next step towards full-stack developing and I am proud that my first attempt at this has proved to contain all the desired functionality.
